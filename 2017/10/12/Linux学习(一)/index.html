Linux学习(一)

Problems are not stop signs, they are guidelines.

1.创建目录：mkdir /tmp/test
创建文件：touch /tmp/test.txt
touch -r file1 file2(将file2的时间记录改成和file1一样)

2.删除文件：rm /tmp/test.txt （-f 没有提示信息）
删除文件夹：rm -r /tmp/test.txt （-f 没有提示信息） -rf递归强制删除

3.复制文件：cp /tmp/res.txt /tmp/des.txt
复制文件夹和文件夹中的内容: cp -r dir/ /tmp/（ ：所有文件和文件夹 ？：一个字符）

4.移动文件 mv /tmp/res.txt /tmp/des.txt

5.其它常用命令

  1. nautilus 打开可视化文件管理
  2. cd .. 返回父目录 cd 回到家目录 ~表示家目录
  3. pwd 查看当前目录
  4. ps -ef 查看进程 kill -s 9 进程ID 杀对应进程
  5. sudo mount /dev/sdb3 /medea/wentai/Document 挂载 sudo umount /dev/sdb3 卸载

6.deb包的安装与软件的卸载
安装.deb包 sudo dpkg -i ***.deb
卸载 sudo apt-get autoremove –purge firefox 注：firefox为待卸载的软件名称
dpkg –get-selections | grep ‘软件相关名称’ 若执行卸载时未知具体软件名称则可执行此行
注意：purge前为两个”-“

例子：Ubuntu 16.04下wine QQ使用一段时间后突然无法启动，先卸载再重装
安装Wine QQ需要先安装Wine

step1.找到安装版本qq的文件夹，比如/opt/longene/qq
step2.打开里面的qq.sh文件，查看你安装版本的qq名package_name = wine-qq7.8-longeneteam
step3.卸载sudo dpkg –purge wine-qq7.8-longeneteam 注意”–purge”是两个短横线”-“

32bit系统直接安装: sudo dpkg -i WineQQ7.8-20151109-Longene.deb 确保当前目录已经下载Wine QQ的deb安装包
64bit先安装32位兼容库 sudo apt-get install libgtk2.0-0:i386 再使用上面的安装指令

（2）进入到解压后的目录下：cd example-1.2.3-1
（3）再依次执行：

8.安装tar.gz(tgz)包
（1）解压tar.gz包：tar -zxvf example-1.2.3-1.tar.gz
（2）后方法和example-1.2.3-1.tar包一样

9.安装.tar.bz2包
安装方法一：
（1）先解压tar.bz2包，执行 bunzip2 example-1.2.3-1.tar.bz2命令解压得到tar包。
（2）后方法和example-1.2.3-1.tar包一样
安装方法二：（推荐）
直接执行 tar -jxvf example-1.2.3-1.tar.bz2命令解压即可

PS：tar命令的两个参数解释：
-z ：是否同时具有 gzip 的属性？亦即是否需要用 gzip 压缩
-j ：是否同时具有 bzip2 的属性？亦即是否需要用 bzip2 压缩

12.本地文件与服务器互传(Linux下命令操作)
本地hello.png文件上传服务器 scp /home/wentai/桌面/hello.png root@172.19.2.7:/var/www/html/pic
服务器文件传到本地文件夹下 scp root@172.19.2.75:/home/root /home/daisy/full.tar.gz
服务器文件传到本地文件夹下未测试,上面命令是否为吧服务器/home/root文件打包后存入本地
还是将服务器上的压缩文件存到本地/home/root/下

13.Ubuntu系统如何更改用户密码
在命令行中输入 passwd 用户名，比如用户名叫joe，那就输入 passwd joe，然后按回车键,
在弹出的界面按照提示输入原来的密码，系统确认后，再按照提示输入新密码两次，结束后按enter即可

14.Ununtu用户管理
