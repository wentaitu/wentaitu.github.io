显而易见，程序是为了使rear指针变量指向另外一块内存区域
运行结果如下：

当输出指针前后的地址和指向的内存区域值时，指针前后并没有变，
一看变换函数参数的确是指针，按地址传递应该不会错。

如果仔细一看会发现，这根本不是按址传递，而是按值传递
因为传入的是指针变量而不是指针的地址(指针除了指向一块内存区域外，自己也有一个地址)，
所以传入的仅仅是指针变量的一个拷贝
解决方法如下：
变换函数参数使用二级指针变量，调用时直接传入指针变量的地址(注意不是传入指针指向的内存区域的地址)
