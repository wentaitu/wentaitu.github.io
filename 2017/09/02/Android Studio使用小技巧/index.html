如何修改应用包名

在Android项目结构右上角setting里面，把 compact empty middle packages 这个打勾的选项去掉即可看见包的层次结构分开，再点击相应包名按照提示修改，

注意：由于包名存在于每个文件中，AS将会自动修改，但仍有例外，例如app目录下build.gradle中的applicationId未修改等等，视编译错误修改文件包名即可
