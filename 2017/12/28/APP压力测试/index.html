Python环境

一 打开手机USB调试
二 确认手机和电脑已经成功连接 adb devices
三 安装待测APP(未安装情况下) adb install app_name.apk
四 发送压力指令 adb shell monkey 1000 工具名称monkey 执行1000条指令(随机)
五 获取App包名 adb logcat | grep START 中间竖线代表管道,把前一个命令的输出作为下一个命令的输入
这里把logcat输出给下一个命令,抓取有START标签的log信息,再打开待测APP即可
六 给指定包打压力 adb shell monkey -p package 1000

Monkey高级参数应用

1.指定事件间的间隔 throwttle参数
adb shell monkey --throttle <milliseconds>
// adb shell monkey --throwttle 1000 10 有10个事件间隔1秒发生

2.指定随机生成数的seed值 seed参数
adb shell monkey -s <seed> <event-count>
// adb shell monkey -p com.wentai.calculator -s 100 50 值seed为100,执行50个事件
以计算器为例 清除计算器历史记录 退出 杀进程 执行monkey指令 两次执行结果一致
故可以进行结果复现
测试时先自定义一个seed值,跑一晚上,第二天早上发现Exception,用相同seed和event-count就可以对结果进行复现
是否为出现Exception的操作才被重现,Excpetion有多次?
未忽略

3.设置触摸事件百分比
adb shell monkey --pct-touch <percent>
// adb shell mobkey --pct-touch 100 180 完成180个纯点击时间
// adb sehll monkey -v --pct-touch 100 180 参数-v会把所有事件详情列出来

4.设置动作事件百分比
adb shell monkey --pct-motion <percent>
// adb shell monkey -v -p com.wentai.calculator --pct-touch 50 --pct-appswitch 40 --pct-motion 10 100
产生100个随机事件,分别占比50%,30%,10%
若只包含某两种或某三种随机事件,相加必须为100,＜100则会产生其它随机事件
未指定忽略异常 执行
adb shell monkey -p com.wentai.calc 1000000
异常即停止,结尾显示seek值,可根据此重现异常
adb shell monkey -p com.wentai.calc -s 14242154 1000000
adb shell monkey -p com.wentai.calc --ignore-crashes 1000000

5.设定轨迹球事件百分比
adb shell monkey --pct-trackball <percent>

6.基本导航事件
设定基本导航事件百分比，输入设备的上 下 左 右
adb shell monkey --pct-nav <percent>

7.主要导航事件
设定主要导航事件百分比,兼容中间键,返回键,菜单按键
adb shell monkey --pct-majornav <percent>

8.系统导航事件-和系统硬件相关
设定系统导航事件百分比,HOME,BACK,拨号,音量键
adb shell monkey --pct-syskeys <percent>

9.启动Activity事件
设定启动pActivity的事件百分比
adb shell monkey --pct-appswitch <percent>

10.设定不常用事件百分比
adb shell monkey --pct-anyevent <percent>

11.忽略崩溃和异常
adb shell monkey --ignorecrashes <event-count>

12.忽略超时事件
adb shell monkey --ignore-timeouts <event-count>

// adb shell monkey -v -p com.wentai.calculator --pct-touch 50 --pct-motion 50 --ignore-crashes --ignore-timeouts 100

DispatchTrackball(long downtime, long eventide, int action, float x, float y, float pressure, float size, int metastate, float xprecision, float yprecision, int device, int edgeflags)
