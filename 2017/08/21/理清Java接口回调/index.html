Do not for one repulse,give up the purpose that you resolved to effect. –Shakespeare

1.简介

  • 回调(callback)是一种常见的程序设计模式．类A调用类B中的方法C，而类B又反过去调用类A中的方法D，这里D称为A的回调方法。
  • 举一例子：顾客是A，厨师是B。面粉是接口Callback。面粉的吃法D是在顾客A的手中，厨师B有包子的做法C。现在A想吃包子，但是A不会做包子，就去找厨师，让厨师B调用方法C做包子，A就把面粉Callback给了厨师，至于厨师怎么做，A不管，A说：只要你做好后给我结果（包子）就OK。至于包子怎么吃，我是直接吃，还是蘸点醋吃，那是A的事了（A怎么吃的过程就是方法D里具体执行的代码），D就是回调方法，最后还是回来调用了自己吃包子的方法．
