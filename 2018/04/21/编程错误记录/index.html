编程错误记录

1.java中字符串比较 == 与 equals() 使用错误

2.控件初始化,findViewById()参数调用xml中控件名错误,如Switch控件却调用了Button控件

3.服务开启后用Switch监听,能开启却无法关闭,子线程在OnCreate中创建并执行任务,应在onStartCommand()中。。。？，开启了子线程，服务关闭而子线程未关闭

4.RecyclerView 重新设置了Adapter于外部,Intent重新设置了于外部,使用时却仍未改
StartActivity(new Intent(….)) 应改为 startActivity(i);Adapter同理

5.汇编代码mov指令写成了move指令

6.自定义Application类后未在清单文件中定义name属性

7.RecyclerView Item中加载图片,若图片过大可能造成内存溢出OOM,造成程序崩溃,应压缩图片

8.调试APP,需要网络连接时未加入网络权限 或 使用模拟器电脑未联网

9.二叉树，创建二叉树时对creat函数返回的root指针未赋值给二叉树类的成员变量root，改为BinTree() { root = creat(root); }

10.语句结束未加分号; 结构体定义为BiNode，使用时用成BinNode; 引用库#include 直接使用cout,应该使用std::cout或添加using namespace std;

11.高德地图API每个包名对应一个key,若修改包名则会无法定位

12.Android网络请求为添加网络权限 请求地址字符串未加http://

13.Git上传到github时,软件代码中包含自己打包的库,库中包含.git文件夹,此库中的内容不会一并上传至github,导致编译错误

14.二分查找递归算法，递归语句应该return xxx自身()，而未在调用自身函数时加return

15.直接使用编辑器写Java代码,String获取长度为str.length(),数组获取长度为arr.length
