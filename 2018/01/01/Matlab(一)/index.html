MATLAB(一)

一个从未犯错的人是因为他不曾尝试新鲜事物。

1、MATLAB视频教程http://www.51zxw.net/list.aspx?cid=456 已看完第一章1.4
2、《Matlab2013实用教程》ISBN 97870406542 看至p0

一、安装

R2017b_win64_dvd1.iso
R2017b_win64_dvd2.iso
两份文件解压放在一起

2017代表年份，安装b版，由于b版修复了a版bug，点击setup.exe
网上搜密钥，并输入，开始安装
安装完毕，打开安装目录，仅有R2017b一个文件夹
1.MATLAB R2017b Win64 Crack文件夹，包含破解文件，直接复制其内的R2017b,粘贴到安装的根目录下，与原R2017b合并
2.将MATLAB R2017b Win64 Crack文件夹下license_server.lic和license_standalone.lic复制到R2017b\licenses目录下

找到 …/R2017b/bin/matlab.exe 双击打开，出现闪退
1.复制开始解压的文件到安装的根目录下；
2.配置license_standalone.lic文件环境变量，名为LM_LICENSE_FILE.
找到 …/R2017b/bin/matlab.exe 双击打开，仍然出现闪退
单击matlab.exe 右键以管理员身份运行，启动成功，添加快捷方式到”开始”屏幕，Matlab R2017b运行正常

开始测试,点击右上角新建脚本,输入

点击保存在某个文件夹，点击运行，提示路径错误，按提示更改，运行正常

测试：命令窗口中输入
3 + 5
按下回车，显示计算结果

测试：命令窗口中输入
r = 2;
回车不会进行输出，继续输入
s = pi * r^2
回车后，显示圆面积值

显示数据为了简洁进行了四舍五入，但计算时用的精确值，也可以修改显示方式
点击最左下角选择Parallel preferences->MATLAB->命令行窗口->可选择不同数值格式->重启MATLAB 永久生效
用命令方式修改：输入format long 本次运行都以long的精度显示，重启MATLAB后恢复原始精度显示

clc清屏
紧凑方式显示结果format compact,同理可在命令窗口选项中数值显示切换为compact然后重启MATLAB

命令历史窗口 记录命令窗口中输入的所有命令，可以将命令重新执行
当前目录 若未给变量指定保存目录则会保存到当前目录中 // save 变量名

工作空间 对MATLAB中涉及的变量进行观察、保存、编辑、提取 右键可保存变量等等
clear 变量名 //删除相应变量
清空全部工作空间 clear或者clear all
建议打开MATLAB首先输入clear
b = [1, 2, 3; 4, 5, 6]
可以双击变量打开变量编辑器，手动输入
点击变量，右键选择plot可以对变量进行绘图
who命令，系统会将定义的变量列举出来，详细查看输入whos
查看某一变量 whos 变量名

编辑调试
数组编辑器

Matlab中的文件类型：m文件，mex文件(可执行文件，可脱离Matlab运行)、fig文件(图像)、mdl文件(模型文件Simulink)、s文件(Simulink)、mat文件(标准数据格式文件)
Matlab中对数据文件.mat的加载：load 即load 数据文件名
打开.m文件直接输入文件名即可不用加扩展名
输入edit 打开编辑窗口
figure 打开图像窗口
guide 打开相应GUI窗口
